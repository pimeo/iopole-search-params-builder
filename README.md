# IOPOLE Query Builder

## Requirements

- Node>=24

## Why this tool?

To perform directory searches, [PA (ex-PDP) IOPOLE search API has a powerful and complex parameter search mechanism](https://docs.ppd.iopole.fr/docs/iopole-api/search). However, the company does not provide a tool to simplify the creation and chaining of parameters yet.

Here is a tool to easily perform an advanced parameter search.

You are free to use and modify it.
